# Icons Directory

This directory contains application icons and graphics.

## ProMirrorGolf_App_Icon.png

The main application icon for ProMirrorGolf. This icon is:
- Displayed in the application window title bar
- Shown in the top-left corner of the GUI header
- Used as the application icon in the taskbar (on supported platforms)

## Usage

The icon is automatically loaded by `main.py` when the application starts. Ensure the file is named exactly `ProMirrorGolf_App_Icon.png` and placed in this directory.

## File Format

- Format: PNG
- Recommended size: 256x256 pixels or larger
- Background: Transparent or dark theme compatible
