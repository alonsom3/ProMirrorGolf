# Icons Directory

This directory contains the ProMirrorGolf application icon.

## ProMirrorGolf_App_Icon.png

The main application icon for ProMirrorGolf. This icon is:
- Displayed in the application window title bar
- Shown in the top-left corner of the GUI header
- Used as the application icon in the taskbar (on supported platforms)

The icon is automatically loaded by `main.py` when the application starts.
